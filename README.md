# STM32 智能人体感应延时照明 + 过热保护综合系统

本项目是一个基于 STM32F103 的嵌入式综合实践项目，实现“暗光环境 + 检测到人体”自动照明，并加入 PWM 调光、延时关灯、过热保护、OLED 显示、蓝牙控制、EEPROM 掉电保存和串口调试等功能。

## 项目功能

- 红外人体感应：使用 PA12 外部中断检测人体靠近。
- 光线检测：通过 ADC 采集光敏传感器数据，判断当前是否为暗光环境。
- 温度检测：通过 ADC 采集芯片温度，用于过热保护。
- 自动照明：只有暗光且检测到人体时，才自动开启灯光。
- PWM 调光：使用 PB10 / TIM2_CH3 输出 PWM，控制 LED1 亮度。
- 延时关灯：人体离开后开始倒计时，到时间后自动关灯。
- 过热保护：温度过高时强制关灯，并通过 PC6 蜂鸣器持续报警。
- 按键设置：支持切换 OLED 页面、修改参数、查看和清零亮灯次数。
- OLED 显示：显示光线值、温度、亮度、剩余时间、系统状态、亮灯次数和 EEPROM 记录数。
- 蓝牙通信：支持手机端查看状态、修改参数、手动开关灯和读取历史记录。
- EEPROM 存储：保存光线阈值、延时时间、默认亮度、亮灯次数和最近 10 条运行记录。
- 串口调试：通过 USART1 输出系统运行日志。

## 主要硬件模块

- STM32F103VETx / STM32F103VGTx 开发板
- KY-032 红外人体感应模块
- 光敏传感器
- STM32 内部温度传感器
- LED13 / PWM 调光灯
- OLED 显示屏
- AT24C128 EEPROM
- 蓝牙串口模块
- 蜂鸣器
- 板载按键

## 关键引脚

| 功能 | 引脚 / 外设 |
|---|---|
| 红外人体感应 | PA12 / EXTI12 |
| 光线 ADC | PC5 / ADC1_IN15 |
| 温度 ADC | ADC1 内部温度通道 |
| PWM 调光 | PB10 / TIM2_CH3 |
| 灯光使能 | PD14 |
| 蜂鸣器 | PC6 / TIM3_CH1 |
| OLED 软件 I2C | PC0 / PC1 |
| EEPROM 软件 I2C | PE3 / PE4 |
| 蓝牙串口 | USART2 |
| 调试串口 | USART1 |

## 系统状态

程序使用三个主要状态：

```c
typedef enum
{
    SYS_STANDBY = 0,
    SYS_LIGHTING,
    SYS_OVERHEAT
} SystemState_t;
```

- `SYS_STANDBY`：待机状态，等待暗光和人体触发。
- `SYS_LIGHTING`：照明状态，灯光开启并进行延时计时。
- `SYS_OVERHEAT`：过热保护状态，强制关灯并蜂鸣器报警。

## 默认参数

默认参数定义在 `Core/Src/app.c` 文件开头：

| 参数 | 默认值 | 说明 |
|---|---:|---|
| 光线阈值 | 3000 | 小于该值认为是暗光 |
| 延时关灯时间 | 40 秒 | 人体离开后的关灯倒计时 |
| 默认亮度 | 70% | 自动开灯时的默认亮度 |
| 过热报警温度 | 35.0 ℃ | 达到后进入过热保护 |
| 过热恢复温度 | 27.0 ℃ | 低于后解除过热保护 |

## 蓝牙指令

手机 APP 或串口助手通过蓝牙发送以下文本命令：

| 指令 | 作用 |
|---|---|
| `GET` | 查询当前光线、温度、亮度、状态、次数等信息 |
| `SET LIGHT=xxx` | 设置光线触发阈值 |
| `SET DELAY=xxx` | 设置延时关灯时间，单位秒 |
| `SET BRIGHT=xxx` | 设置默认亮度，范围 0~100 |
| `ON` | 手动开灯 |
| `OFF` | 手动关灯 |
| `CLEAR` | 清零感应亮灯次数 |
| `GET D` | 开启实时数据上报 |
| `GET E` | 停止实时上报，并返回 EEPROM 中最近 10 条记录 |

## STM32CubeMX / Keil 配置要点

需要开启或配置：

- `ADC1`
  - PC5 配置为 `ADC1_IN15`
  - 开启内部温度传感器通道
- `TIM2`
  - 配置为 PWM
  - 通道：`TIM2_CH3`
  - 输出引脚：PB10
- `TIM3`
  - 配置为 PWM
  - 通道：`TIM3_CH1`
  - 输出引脚：PC6，用于蜂鸣器
- `USART1`
  - 用于电脑串口调试
  - 波特率：115200
- `USART2`
  - 用于蓝牙模块通信
  - 波特率：115200
- `GPIO EXTI`
  - PA12 配置为外部中断
  - PB5 / PB6 / PB7 / PC2 配置为按键外部中断
- `GPIO Output`
  - PD14 用于灯光使能
  - PC0 / PC1 用于 OLED 软件 I2C
  - PE3 / PE4 用于 EEPROM 软件 I2C

## 使用方法

1. 使用 Keil 打开：

   ```text
   MDK-ARM/STM32_SmartLight_Protect.uvprojx
   ```

2. 编译工程，生成 HEX 文件。

3. 使用 ST-LINK 下载到开发板。

4. 打开电脑串口助手，连接 USART1，波特率设置为 `115200`。

5. 使用手机蓝牙串口工具连接蓝牙模块，发送 `GET`、`ON`、`OFF`、`SET DELAY=20` 等命令测试功能。

## 测试建议

- 遮挡光敏传感器，观察光线值是否降低并触发暗光判断。
- 靠近红外模块，观察是否自动开灯。
- 离开红外检测范围，观察是否开始倒计时并自动关灯。
- 发送 `SET BRIGHT=30`，观察 PWM 亮度变化。
- 调低过热阈值进行测试，观察是否进入过热保护并蜂鸣器报警。
- 修改参数后断电重启，确认 EEPROM 是否能恢复上次设置。
- 发送 `GET D` 查看实时上报，发送 `GET E` 查看 EEPROM 最近 10 条记录。
